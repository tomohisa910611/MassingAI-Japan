# 次段階のボリューム計算ルール

この資料は、次に実装する法規・ボリューム計算画面の仕様です。現在の画像読取画面には、まだこの計算機能を追加していません。

## 1. 表示の基本方針

最初の結果は、安全側となる「緩和なし」で表示します。同じ画面に、条件をすべて満たした場合の「最大緩和参考値」を分けて表示します。

| 表示 | 用途 |
|---|---|
| 緩和なし（初期表示） | 初期検討で使う厳しい条件 |
| 最大緩和参考値 | 事業性を確認するための上限側シナリオ |

最大緩和参考値には「役所確認前」「条件付き」と明記します。緩和なしの数値と混ぜません。

## 2. この敷地の初期入力

- 敷地面積：92.34㎡
- 前面道路幅員：7.970m
- 指定容積率：600%
- 道路幅員による基準容積率：`7.970 × 0.6 × 100 = 478.2%`
- 指定建蔽率：80%
- 地区区分：B地区の可能性が高い（役所確認前）
- 壁面位置の制限：0.5mの可能性が高い（役所確認前）

## 3. 緩和なしの初期結果

すべての用途で、最初は次の条件を使います。

- 容積率：478.2%
- 延べ面積上限の目安：441.6㎡
- 建蔽率：80%
- 建築面積上限の目安：73.87㎡
- 道路斜線：通常の道路斜線を適用
- 地区計画による容積率割増：適用しない
- 地区計画による道路斜線緩和：適用しない
- 防火地域による建蔽率加算：適用しない

これは法的な最終値ではなく、初期比較用の安全側の値です。

## 4. 壁面後退の考え方

理解は「条件付きで正しい」です。

計画図上で敷地前面に壁面位置の制限が指定されていれば、最低限の壁面後退は必須です。この敷地では0.5mの可能性が高いものの、正式計算前に千代田区の都市計画情報提供ポータルで確認します。

壁面後退を0.5m、0.75m、1.0mと大きくすると容積率が変わるのは、地区計画の容積率緩和を使う場合です。緩和を使わない設定では、壁面後退を増やしても容積率478.2%は変えません。ただし、建築できる平面形状や通常の道路斜線には影響します。

| 壁面後退 | 緩和を使う場合の加算 | 条件付き容積率 |
|---:|---:|---:|
| 0.50m以上0.75m未満 | +60% | 538.2% |
| 0.75m以上1.00m未満 | +90% | 568.2% |
| 1.00m以上 | +120% | 598.2% |

地区計画は、幅員4mを超える道路では1m後退を誘導しています。ただし「誘導する1m」と「計画図で指定された最低壁面線」は同じ意味ではありません。

## 5. 用途別の3パターン

| 用途 | 緩和なし | 最大緩和参考値 | 主な注意点 |
|---|---:|---:|---|
| 住宅・マンション | 478.2% / 約441.6㎡ | 暫定598.2% / 約552.4㎡ | 1m以上の後退などが必要。基準容積率超過部分は住宅等。対象住戸は原則40㎡以上などの条件あり |
| 商業 | 478.2% / 約441.6㎡ | 純商業は暫定478.2% | 基準容積率を超える部分は住宅等に限定されるため、純商業では割増を見込まない |
| ホテル | 478.2% / 約441.6㎡ | 暫定478.2% | ホテルは割増対象用途として明記されていないため、安全側では対象外。千代田区へ確認が必要 |

商業またはホテルに住宅を組み合わせる複合用途は、住宅部分の割合に応じた別計算が必要です。最初の3パターンでは、比較を分かりやすくするため単一用途として計算します。

## 6. 将来の画面に必要な項目

1. 用途選択：住宅・マンション／商業／ホテル
2. 計算モード：緩和なし／最大緩和参考
3. 壁面後退：0.5m／0.75m／1.0m
4. 結果：容積率、延べ面積、建蔽率、建築面積、道路斜線、高さ制限
5. 未確認条件：B地区、壁面線、道路情報、用途解釈
6. 根拠資料へのリンク

初期値は必ず「緩和なし」にします。「最大緩和参考」を選んだときだけ条件付きの割増値を表示します。
